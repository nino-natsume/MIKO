## Setup instructions

Test Environment: 1 panel + PHP 7.4 + MySQL 5.6

The following three files need modification:
Installation directory/config/config_global.php
Installation directory/config/config_ucenter.php
Installation directory/uc_server/data/config.inc.php

Using config_global.php as an example, modify other files accordingly.

The dbhost column in the table below should be based on your actual connection information. If it's a 1 panel, the database container name to modify is the content entered here.

| Category | Settings (Modify the content within '') |
| :---: | :---: |
| dbhost | 'mysql' |
| dbuser | 'ciallo' |
| dbpw | '0d000721' |
| dbname | 'ciallo' |
