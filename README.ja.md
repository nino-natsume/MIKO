## セットアップ手順

テスト環境: 1パネル + php7.4 + mysql5.6

以下の3つのファイルを修正する必要があります。

インストールディレクトリ/config/config_global.php インストールディレクトリ/config/config_ucenter.php

インストールディレクトリ /uc_server/data/config.inc.php

config_global.php ファイルを例として、他のファイルも適宜変更してください。

以下の表の dbhost 列は、実際の接続情報に基づいて入力してください。パネルが 1 つの場合は、変更するデータベースコンテナ名がここに入力された内容になります。

| カテゴリ | 設定 ('' 内の内容を変更してください) |

| dbhost | 'mysql' |
| dbuser | 'ciallo' |
| dbpw | '0d000721' |
| dbname | 'ciallo' |
